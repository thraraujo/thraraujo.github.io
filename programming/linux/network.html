MAC Address

MAC means Medium Access Control.

I can see the MAC address of my device with:

$ ifconfig # see the field *ether* entry. $ ip link show $ nmcli device show | grep GENERAL.HWADDR $ cat /sys/class/net/*/address # this several addresses $ cat /sys/class/net/wlp0s20f3/address # this seems to be the most convenient

In order to check my network, I install the util `nmap`, and I scan my network

$ sudo nmap -sP 192.168.15.0/24 $ arp -a # I can check the MAC addresses of devices connected to my network.
