Some Useful Git Commands

Push a tag to remote


                git push origin 

                # not recommended
                git push --tags

See more: push a tag to a remote

Tag to older commit


                git tag v1.2 9fceb02

                # with message 
                git tag -a v1.2 9fceb02 -m "Message here"

See more: push a tag to a remote

Forget deleted branches


                git remote prune origin

See more: push a tag to a remote
