Org-plot

Org-plot is a nice emacs tool to plot some graphs in an org file directly from a table.

First thing to do is to install gnuplot

sudo apt install gnuplot # Debian/Ubuntu sudo dnf install gnuplot # Fedora

I added the following lines to my config:

;; Orgplot (use-package gnuplot) (autoload 'gnuplot-mode "gnuplot" "Gnuplot mode" t) (setq auto-mode-alist (append '(("\\.plt\\'" . gnuplot-mode)) auto-mode-alist))

After that, we run

M-x org-plot/gnuplot

or C-c " g.
